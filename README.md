## Installation

Using packer:

```lua
use({
  'nvim-neotest/neotest',
  requires = {
    'amar-laksh/neotest-catch2',
  }
  config = function()
    require('neotest').setup({
      ...,
      adapters = {
        require('neotest-catch2'),
      }
    })
  end
})
```
